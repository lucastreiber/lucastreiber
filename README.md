## 👋 À propos de moi
Je suis un développeur passionné par les jeu vidéo et de la manière dont ils sont créé. 
Quand j'ai du temps libre j'aime faire des minis projet en rapport avec le jeu vidéo.
Ce que je trouve le plus fun c'est après avoir coder une fonctionnalité pendant 1 heure que ça marche parfaitement sans aucun bug.
